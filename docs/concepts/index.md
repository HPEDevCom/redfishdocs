# Redfish at a glance

This is a placeholder for presenting Redfish. Something like:

Redfish is a programming interface enabling state-of-the-art server or appliance management. The Redfish API uses the basic HTTP operations (GET, PUT, POST, DELETE, and PATCH) to submit or return a JSON formatted resource to or from a URI on iLO 5.

With modern scripting languages, you can easily write simple REST clients for RESTful APIs. Most languages, like Python, can transform JSON into internal-data structures, like dictionaries, allowing for easy access to data.
